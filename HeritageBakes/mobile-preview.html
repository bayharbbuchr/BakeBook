Mobile Testing Instructions

To test the mobile responsiveness of the HeritageBakes application, follow these instructions:

Option 1: Use Browser Developer Tools (Recommended)

  1. Open the application at http://localhost:5000
  2. Right-click anywhere on the page and select "Inspect" or press F12
  3. Click on the "Toggle device toolbar" button (looks like a phone/tablet icon) or press Ctrl+Shift+M
  4. Select a mobile device from the dropdown at the top of the viewport (e.g., iPhone X, Galaxy S9)
  5. Test the responsive features by navigating through the application

Option 2: Resize Browser Window

  1. Open the application at http://localhost:5000
  2. Manually resize your browser window to be narrow (less than 640px wide)
  3. Test the responsive features as the layout adapts to the smaller viewport

Key Mobile Features to Test

  • Mobile Navigation: The hamburger menu should appear and be functional on small screens
  • Responsive Layout: Content should reflow to a single column on mobile
  • Touch-friendly: All buttons and interactive elements should be large enough for touch
  • Font Sizes: Text should be readable without zooming
  • Forms: Input fields should be properly sized for mobile input
  • Floating Action Button: The add recipe FAB should appear on mobile views
